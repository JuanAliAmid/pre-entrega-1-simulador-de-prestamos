Simulador de prestamos

Descripción:
Este proyecto es un simulador de prestamos. Permite registrar prestamos, calcular intereses según la cantidad de cuotas, listar prestamos otorgados, buscar por nombre de cliente y calcular el total general.

Funcionalidades:
-Agregar prestamo con nombre, monto y cantidad de cuotas.
-Cálculo de intereses según la cantidad de cuotas ingresadas.
-Listar los prestamos registrados.
-Calcular el total general de los prestamos.
-Los prestamos se almacenan en localStorage para mantener la informacion aunque se recargue la pagina.
-Uso del DOM para mostrar y actualizar los datos en pantalla.

Tecnologias utilizadas:
-HTML
-JavaScript
-DOM
-localStorage
